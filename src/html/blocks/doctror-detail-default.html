Опыт работы

С 1989 по 1991 год клинический ординатор в терапевтических отделениях Свердловской областной клинической больницы № 1.

С 1991 по 1999 год работал врачом лечебной физкультуры и врачом-терапевтом Свердловского областного клинического психоневрологического госпиталя для ветеранов войн.

С 1999 по 2006 год врачом-кардиологом Свердловской областной клинической больницы № 1.

С 2006 по 2011 год – начальник отдела телемедицины Управления инновационных информационных технологий Уральской государственной медицинской академии, по совместительству ассистент кафедры пропедевтики внутренних болезней (2006-2009 гг), доцент кафедры терапии факультета повышения квалификации и профессиональной переподготовки (2008-2014 гг).

С 2011 по 2017 год работал врачом-кардиологом ООО «Городская больница № 41».

В 2017-2018 годах – врачом-кардиологом, терапевтом и врачом функциональной диагностики ООО «Преображенская клиника» и ООО «Международный медицинский центр Европа-Азия».
